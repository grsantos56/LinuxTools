# Linux Tools

O **Linux Tools** é um gerenciador de pacotes criado para facilitar a instalação de programas comumente usados em ambientes Linux. Conta com ferramentas de **desenvolvimento**, **internet**, **design e multimídia**, **hacking** e muito mais, todos disponíveis para instalação com apenas um clique, automatizando tarefas e tornando a experiência mais prática para desenvolvedores e usuários em geral.

---

### Lista de programas disponíveis para instalação:

---

### 🧱 1. Desenvolvimento

#### Linguagens de Programação
- `python3`, `python3-pip` – Linguagem e gerenciador de pacotes Python.
- `openjdk-8-jdk`, `openjdk-17-jdk`, `default-jdk` – Kits de desenvolvimento Java (diferentes versões).
- `nodejs`, `npm` – Ambiente JavaScript e seu gerenciador de pacotes.
- `gcc`, `g++` – Compiladores C e C++.
- `ruby-full` – Ambiente completo para Ruby.
- `golang` – Linguagem Go.
- `rustc`, `cargo` – Compilador e gerenciador de pacotes Rust.

#### 🔧 Frameworks
- `django` – Framework web para Python.
- `flask` – Microframework web para Python.
- `springboot` – Framework web para aplicações Java.
- `react` – Biblioteca JavaScript para interfaces web.
- `angular` – Framework frontend baseado em TypeScript.
- `laravel` – Framework PHP para aplicações web.

#### 🧠 Compiladores e Ferramentas de Build
- `build-essential` – Conjunto essencial de compiladores e ferramentas.
- `cmake` – Sistema de build multiplataforma.
- `make`, `ninja-build` – Ferramentas de automação de compilação.

#### 🧪 Testes e DevOps
- `docker.io`, `docker-compose` – Contêineres e orquestração leve.
- `ansible` – Automação de configuração.
- `vagrant` – Criação e gerenciamento de máquinas virtuais.

#### 🧰 Ferramentas CLI
- `tmux` – Gerenciador de janelas no terminal.
- `curl`, `wget` – Ferramentas para requisições HTTP/FTP.
- `gh` – Interface de linha de comando para o GitHub.
- `jq` – Manipulação de JSON via terminal.
- `httpie` – Requisições HTTP amigáveis para APIs.

#### 📝 IDEs e Editores
- `vim`, `neovim` – Editores de texto no terminal.
- `code` – Visual Studio Code, editor moderno e extensível.
- `geany` – IDE leve para múltiplas linguagens.
- `jetbrains-toolbox` – Instalador e atualizador de IDEs JetBrains.

#### 🗃️ Banco de Dados
- `sqlite3`, `sqlitebrowser` – Banco de dados leve e seu gerenciador gráfico.
- `mysql-server`, `mysql-client` – Banco de dados MySQL/MariaDB e cliente.
- `postgresql`, `pgadmin4` – Banco de dados PostgreSQL e interface gráfica.
- `redis-server` – Banco de dados chave-valor em memória.

### 🕵️ 3. Hacking

- `nmap` – Varredura de redes e portas.
- `whois` – Consulta a dados de registro de domínios.

#### 🌐 Análise de Tráfego
